# DataVision
DataVision is a data visualization tool that helps you analyze data structure, clean the data and plot trends and quality.

The backend is built with Java 21 and Spring Boot with microservices architecture in Python.

## Tech Stack
- Java 21
- Spring Boot
- Python
- Docker
- PostgreSQL
- MinIO

## Features
- **JWT Authentication** - Secure login with token-based authentication.
- **File upload with MinIO** - Upload files to MinIO.
